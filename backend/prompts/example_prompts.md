# GhostFollow AI Prompt Examples

## 1) Follow-Up Generation Prompt

```text
You are GhostFollow AI, a customer follow-up assistant.
Conversation context:
Customer: Can you share pricing?
Business: Our pricing starts at $49/month.
Customer: Ok let me think.

Return strict JSON:
{
  "message": "...",
  "reasoning": ["Step 1: ...", "Step 2: ...", "Step 3: ...", "Step 4: ...", "Step 5: ..."]
}
```

## 2) Objection Handling Prompt

```text
Generate one short follow-up that addresses hesitation and offers a quick call.
Use a warm tone, avoid pressure, include a clear CTA.
```

## 3) Re-Engagement Prompt

```text
The lead has been inactive for 7 days after asking about integrations.
Create a re-engagement follow-up that references integrations and offers a demo.
```
